Learn **Pandas step by step** and master data analysis using Python with practical examples and real-world datasets.

This repository is designed to help you **quickly understand and apply Pandas concepts** through structured modules, hands-on coding, and real-world scenarios.

> 🚀 No unnecessary theory — only what you need to become confident in data analysis.

## 📌 What You'll Learn

This course takes you from **basics to advanced Pandas concepts** in a structured way:

| Module | Description |
|------|------------|
| Module 1: Introduction to Pandas | Understand what Pandas is and why it is used |
| Module 2: Core Data Structures | Learn Series and DataFrame |
| Module 3: Data Loading | Read CSV, Excel, JSON files |
| Module 4: Data Inspection | Head, tail, info, describe |
| Module 5: Data Selection | Indexing, slicing, filtering |
| Module 6: Data Cleaning | Handle missing values, duplicates |
| Module 7: Data Manipulation | Add, update, delete columns |
| Module 8: GroupBy Operations | Aggregations and analysis |
| Module 9: Merging & Joining | Combine multiple datasets |
| Module 10: Time Series | Work with date and time data |
| Module 11: Real-world Practice | Solve practical datasets |

## ▶️ Getting Started

### ✅ Prerequisites

- Basic Python knowledge  
- Python installed (3.8+)  
- Jupyter Notebook / VS Code  

---

### ⚙️ Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/samanwaya938/pandas_tutorial
cd pandas_tutorial
```

2. Install required libraries

```bash
pip install pandas numpy
```

3. Start practicing

```bash
jupyter notebook
```
